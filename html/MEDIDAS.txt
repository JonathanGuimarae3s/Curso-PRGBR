EM: É RELATIVO AO ELEMENTO PAI COMO O PORCENTO

IV {
      WIDTH:300px
      HEIGHT:300px
}
P{
WIDTH:1EM
HEIGHT: 2EM
}
/* A TAG P TERÁ 1EM = TAMANHO DO ELEMENTO PAI,
LOGO TERÁ 300PX DE ALTURA E 600PX DE LARGURA*/



VW : É REFERENTE A LARGURA DA NOSSA JANELA 1VW = 1 % DA NOSSA JANELA


VH : É REFERENTE A ALTURA DA NOSSA JANELA 1VH = 1 % DA NOSSA JANELA

VMIN : É REFERENTE AO VALOR MINIMO DA JANELA 
VMAX :  É REFERENTE AO VALOR MAXIMO DA JANELA 
 
%: É REFERENTE A MESMA PROPIEDADE DO ELEMENTO PAI
 EX:

<DIV><P></P></DIV>

DIV {
      WIDTH:300px
      HEIGHT:300px
}
P{
WIDTH:50%
HEIGHT: 50 %
}
/* A TAG P TERÁ 50% DA ALTURA E DE LARGURA DA TAG DIV  QUE´POSSUI 300PX,
LOGO TERÁ 150PX DE ALTURA E DE LARGURA*/
